Decoding the Unspoken: Japanese Social Radar V3.0
I’ve always been fascinated by the layers of "Honne" (true feelings) and "Tatemae" (public face) in Japanese communication. 
To map this "Hidden Intent," I built an interactive Sunburst Radar that lets you hear the social subtext.
The Tech Journey:
•	Visuals: 3-tier nested Sunburst in Flourish (Category → Context → Phrase).
•	Audio: Integrated Neural AI voices to capture the "Hidden Intent" and nuance.
•	Engineering: Custom HTML/CSS "Masterpiece" popups for stable audio playback..
This project wasn't just about data; it was about solving the "Page Not Responding" challenges of high-context web design and ensuring a stable,
beautiful user experience.
